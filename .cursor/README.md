# Cursor Chat Organization

## This Project: DIGIfor1Health Web

This workspace is for the DIGIfor1Health Jekyll website project only.

### What belongs here:
- Jekyll/Liquid template questions
- Website styling (SCSS) questions
- Content updates for pages
- Navbar/footer modifications
- Project structure questions

### What doesn't belong here:
- Questions about other projects
- General programming questions
- Questions unrelated to this website

### Best Practice:
- Start a **new chat** for each major task
- Use separate Cursor windows for other projects
- Keep conversations focused on this project
